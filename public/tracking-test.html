Real journey and cleaning test

This uses the Cleaner browser's real geolocation and streams the latest point and cleaning progress to one private Landlord viewer. It is not a booking and it does not contact anyone.

  • Foreground browser use only—background phone tracking is not promised.
  • Only the latest point is held in server memory; no route history is written to disk.
  • The point expires after two minutes, and Stop or Delete removes it immediately.
  • No external map, advertising service or Polsia credit is used.
